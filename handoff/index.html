  width: 100%;

Hero

View on demo →

Background

#F5F5F0 (Cream)

Layout

Two-column Flexbox container: left column 55% (text content), right column 45% (video/image). Stacks vertically on mobile.

  • Left column: H1 heading (two lines — line 1 bold black, line 2 italic light blue), subheading paragraph, CTA note
  • Right column: Background video (hero-video.mp4) with fallback poster image. Overlaid with a white "credential card" positioned bottom-right

Elementor Build

  • Two-column Container widget (55/45 split)
  • Left: H1 Heading widget. For two-tone styling (bold black + italic blue), use <span> inside the heading with custom CSS to color it #0164DB and make it italic + font-weight 300
  • Right: Video widget or Section with background video
  • Credential card: Inner Container with white bg, positioned with z-index. Use an Icon widget for the certificate badge
  • Mobile: Stack order — heading, image, credential card

Spacing

Section min-height: 100vh. Padding: 80px top (desktop) / 128px top (mobile, accounting for fixed nav), 24-48px horizontal.

Animations

Use Elementor: Fade In Up entrance animation on the heading, 700ms. Fade In Right on the credential card. Skip: clip-path text reveal, parallax scroll on the image, hero-zoom keyframe on mobile image.

Elementor's Tabs widget doesn't have a native vertical orientation — use a 2-column Container instead: left column has stacked Button widgets (one per tab), right column shows the active content. Skip: Rotating atom icon background decoration, AnimatePresence cross-fade + blur between tabs (Elementor tabs have their own built-in transition), counter animation (or use Elementor's Counter widget for this).
